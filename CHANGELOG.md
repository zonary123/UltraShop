# Changelog

## [1.0.0] - 2025-12-01

- "CobbleShop" -> "UltraShop" renaming.

### Features

- Added new format cooldown in dynamic shop. The new format is "1y 1mo 1w 1d 2h 30m 15s".
- There is now a checker that looks for a similar UUID in another product to generate a new UUID and avoid cooldown
  conflicts.
- Removed rotation announcement for shops.

### Bug Fixes

- Fixed minor bugs in the initial release.
